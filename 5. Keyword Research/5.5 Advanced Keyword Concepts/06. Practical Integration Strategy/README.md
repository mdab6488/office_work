### Quick Reference Checklist

When creating any piece of content, verify:

**✓ Autocomplete Research**
- [ ] Researched main keyword + a-z variations
- [ ] Identified question words (how, what, why, etc.)
- [ ] Found preposition variations (for, with, without, etc.)

**✓ PAA Integration**
- [ ] Extracted all PAA questions for target query
- [ ] Created FAQ section or H2/H3 subheadings
- [ ] Answered questions concisely (40-60 words)
- [ ] Added comprehensive context after quick answer

**✓ Related Searches Coverage**
- [ ] Listed all related searches from SERP
- [ ] Analyzed related searches for related searches (2nd level)
- [ ] Created internal links to cover gaps
- [ ] Added related topics to content

**✓ Semantic Optimization**
- [ ] Identified 20+ semantically related terms
- [ ] Naturally included synonyms and variations
- [ ] Covered related concepts and subtopics
- [ ] Used topic modeling tool for verification

**✓ Entity Optimization**
- [ ] Defined primary entity clearly
- [ ] Identified related entities (minimum 10)
- [ ] Added structured data markup
- [ ] Created internal entity network
- [ ] Linked to authoritative external sources
- [ ] Used entity-rich anchor text

**✓ Technical Implementation**
- [ ] Schema markup validates in Google's tool
- [ ] Internal links use descriptive, entity-rich anchors
- [ ] Images have entity-relevant alt text
- [ ] URL structure reflects entity hierarchy
- [ ] Meta description includes primary entity
