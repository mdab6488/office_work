## Introduction

On-page SEO audit is the process of analyzing individual web pages to optimize them for search engines and improve rankings. This guide covers the five critical components of on-page optimization with actionable strategies and real-world examples.

### Why On-Page SEO Matters

- **Direct Control**: You have complete control over on-page elements
- **Quick Wins**: Changes can be implemented immediately
- **Foundation**: Essential for technical and off-page SEO success
- **User Experience**: Improves both rankings and user engagement
- **Conversion Impact**: Optimized pages convert better

### Audit Frequency
- **New Sites**: Monthly for the first 6 months
- **Established Sites**: Quarterly comprehensive audits
- **High-Traffic Pages**: Monthly spot checks
- **After Algorithm Updates**: Within 2 weeks

## Content Quality Assessment

Content quality is the foundation of on-page SEO. Google's algorithms increasingly prioritize helpful, relevant, and high-quality content.

### 1. Content Relevance & Search Intent

#### What to Check:
- Does content match the search intent (informational, navigational, transactional, commercial)?
- Does it answer the user's primary question?
- Is it comprehensive enough for the topic?

#### Real-Life Example:

**Bad Example** (Keyword: "best running shoes for beginners")
```

- 300 words listing shoe brands

- No budget recommendations
- Generic affiliate links

```

- Comparison tables with pros/cons
- Video demonstrations

### 5. Content Uniqueness

#### Duplicate Content Check:

**Tools to Use:**
```
1. Copyscape (External duplicates)
2. Screaming Frog (Internal duplicates)
3. Siteliner (Site-wide duplicate check)
4. Google Search Console (Duplicate title/meta tags)
```

**How to Check:**

**Method 1: Manual Google Search**
```
1. Take unique sentence from your content
2. Put in quotes: "exact sentence from your article"
3. Search in Google
4. Check if it appears on other sites

Example:
Search: "The fundamental principle of keyword clustering involves grouping semantically related terms"
Result: Should only show your site
```

**Method 2: Copyscape Premium**
```
1. Enter URL: https://yoursite.com/article
2. Run batch search
3. Review results:

Results Example:
- 0 matches = Unique âœ“
- 1-5 matches = Check if they're citing you
- 5+ matches = Likely plagiarism or syndication issue
```

**Internal Duplicate Issues:**
```
Common Problems:

1. Printer-friendly versions
   Fix: Add canonical tag or noindex

2. Pagination
   Fix: Use rel="next" and rel="prev"

3. Product variations
   Fix: Canonical to main product page

4. HTTP vs HTTPS
   Fix: 301 redirect all HTTP to HTTPS

5. WWW vs non-WWW
   Fix: Choose one, redirect the other

Example Canonical Tag:
<link rel="canonical" href="https://www.example.com/article" />
```

   c) 301 redirect to better page

4. Update regularly

**Issue 2: Duplicate Content**
```
Problem: Same content on multiple URLs
Impact: Diluted rankings, wasted crawl budget

Solution:
1. Identify duplicates (Siteliner, Copyscape)
2. Choose canonical version
3. Add canonical tag to duplicates:
   <link rel="canonical" href="https://example.com/original">

5. Update internal links

- HTTP vs HTTPS
- WWW vs non-WWW
- URL parameters
- Printer-friendly versions
- Product variations
```

After:
"Effective SEO requires multiple strategies working together..."

Bulk Fix Script (WordPress):
```php
// Add to functions.php
function auto_alt_from_title( $attachment_id ) {
    if ( ! get_post_meta( $attachment_id, '_wp_attachment_image_alt', true ) ) {
        $image_title = get_the_title( $attachment_id );
        update_post_meta( $attachment_id, '_wp_attachment_image_alt', $image_title );
    }
}
add_action( 'add_attachment', 'auto_alt_from_title' );
