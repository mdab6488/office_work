
## Integrations & Extensions

### Official Integrations

#### **Google Analytics**
- Import traffic data directly into Moz
- See organic traffic alongside rankings
- Track conversion data per keyword

#### **Google Search Console**
- Import actual Google ranking data
- See impressions and clicks per keyword
- More accurate than third-party tracking

#### **Google My Business** (via Moz Local)
- Manage local listings
- Monitor reviews across platforms
- Update business information instantly

---

### Browser Extensions

#### **MozBar** (Chrome, Firefox)

**What It Does:**
- See DA/PA instantly on any website
- Check page optimization score
- Analyze SERP results with metrics
- Highlight links (follow vs nofollow)
- Export SERP results to CSV

**Real-World Use:**
```
Scenario: Evaluating guest post opportunities

Visit potential blog: www.marketingblog.com
MozBar shows:
- Domain Authority: 47
- Page Authority: 38
- Spam Score: 2%
- Link Type: Follow

Decision: Great guest post opportunity! 
(DA 30+, low spam, dofollow links)
```
