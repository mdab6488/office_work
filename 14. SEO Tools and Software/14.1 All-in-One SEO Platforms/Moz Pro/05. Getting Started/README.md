
## Getting Started

### Step 1: Sign Up for Free Trial
1. Visit moz.com/free-trial
2. Enter email and create password
3. No credit card required (30 days free)

### Step 2: Add Your First Campaign
```
1. Click "Add Campaign"
2. Enter your website URL: www.yoursite.com
3. Choose your primary search engine (Google.com)
4. Select your location/country
5. Add up to 10 competitor URLs
6. Save campaign
```

### Step 3: Add Keywords to Track
```
1. Go to Rankings â†’ Add Keywords
2. Enter 10-50 important keywords
3. Moz will show current rankings
4. Set up weekly ranking reports
```

### Step 4: Run Your First Site Crawl
```
1. Navigate to Site Crawl
2. Click "Start New Crawl"
3. Wait 15-60 minutes (depending on site size)
4. Review critical issues first
5. Create prioritized action plan
```

### Step 5: Research Keywords
```
1. Open Keyword Explorer
2. Enter seed keyword related to your business
3. Analyze search volume and difficulty
4. Export keyword list (CSV)
5. Prioritize keywords with high opportunity
```

### Step 6: Install MozBar Extension
```
1. Go to Chrome Web Store
2. Search "MozBar"
3. Install extension
4. Login with Moz Pro credentials
5. Now see DA/PA on any website you visit
```
