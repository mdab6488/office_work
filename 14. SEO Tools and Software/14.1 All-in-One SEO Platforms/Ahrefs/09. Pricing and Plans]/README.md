
## Pricing and Plans

### Subscription Tiers (2024)

**Lite Plan** - $129/month
- 1 user
- 5 projects
- 500 keywords tracked
- 25,000 results per report
- Good for: Individual bloggers, small businesses

**Standard Plan** - $249/month
- 1 user (3 users available)
- 20 projects
- 1,500 keywords tracked
- 100,000 results per report
- Good for: Growing agencies, marketing teams

**Advanced Plan** - $449/month
- 1 user (5 users available)
- 50 projects
- 5,000 keywords tracked
- 500,000 results per report
- Good for: Large agencies, enterprises

**Enterprise Plan** - Custom Pricing
- Unlimited users
- Unlimited projects
- Custom limits
- API access
- Dedicated support
