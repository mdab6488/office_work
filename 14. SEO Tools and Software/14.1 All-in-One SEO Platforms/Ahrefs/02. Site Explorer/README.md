
## Site Explorer

### What It Does
Site Explorer is your window into any website's organic search performance, backlink profile, and overall SEO health.

### Key Metrics

#### 1. **Domain Rating (DR)**
- **Scale**: 0-100
- **What it measures**: The strength of a website's backlink profile
- **Real-Life Example**: 
  - Amazon.com has a DR of 96
  - A new blog might have DR 0-10
  - An established niche site might have DR 30-50

#### 2. **URL Rating (UR)**
- **What it measures**: The strength of a specific page's backlink profile
- **Example**: 
  ```
  Domain: techcrunch.com (DR 91)
  Specific Article: techcrunch.com/2024/01/ai-trends
  UR might be 45 if that specific page has fewer backlinks
  ```

#### 3. **Organic Traffic**
- Shows estimated monthly search traffic
- **Real Example**:
  ```
  Website: neilpatel.com
  Organic Traffic: ~4.2M visits/month
  Organic Keywords: ~400K keywords ranking
  ```

### Practical Use Case: Competitor Analysis

**Scenario**: You run a fitness blog and want to analyze a competitor.

**Steps**:
1. Enter competitor domain: `menshealth.com`
2. Review their metrics:
   ```
   DR: 88
   Organic Traffic: 12M/month
   Organic Keywords: 850K
   Backlinks: 15.2M
   ```
3. Click on "Top Pages" to see their best-performing content
4. Discover their article "10 Minute Ab Workout" gets 250K monthly visits
5. **Action**: Create better, more comprehensive content on the same topic

### Backlink Analysis Example

**Real-Life Scenario**: You notice your competitor got a backlink from Forbes.

**What You See in Ahrefs**:
```
Referring Domain: forbes.com (DR 95)
Referring Page: forbes.com/health/fitness/best-workout-apps
Anchor Text: "fitness expert advice"
Do-Follow: Yes
First Seen: 2024-10-15
```

**Action Steps**:
1. Analyze why Forbes linked to them
2. Find the journalist's email (Ahrefs can help with this)
3. Pitch your own content to Forbes
4. Or create similar linkable assets
