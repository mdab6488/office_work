
## Domain Overview & Analytics

### What is Domain Overview?

Domain Overview provides a bird's-eye view of any website's online performance, including organic and paid search traffic, backlinks, and top-performing content.

### Real-Life Example 1: Analyzing a Competitor

**Scenario**: You run an e-commerce store selling fitness equipment and want to analyze your competitor "rogue fitness.com"

**Steps**:
1. Enter `roguefitness.com` in the Domain Overview search bar
2. Review the dashboard metrics:

**What You'll See**:
```
Authority Score: 68/100
Organic Search Traffic: 2.1M visits/month
Paid Search Traffic: 45K visits/month
Backlinks: 125K from 3.2K domains
Organic Keywords: 456K keywords ranked
Top Organic Keywords:
  - "weight bench" (112K searches, Position 3)
  - "power rack" (89K searches, Position 2)
  - "barbells" (67K searches, Position 1)
```

**Actionable Insights**:
- They're getting 97.8% traffic from organic (not paid) - focus on SEO
- Their top keywords are product-focused - create better product pages
- Authority Score of 68 means they have strong backlinks - you need link building

### Real-Life Example 2: Monitoring Your Own Website

**Scenario**: You manage `myhealthblog.com` and want to track monthly performance

**What to Monitor**:
```
Month-over-Month Changes:
- Organic Traffic: +15% (from 50K to 57.5K)
- Ranking Keywords: +234 new keywords
- Top Content: "Keto Diet Guide" (12K visits)
- Traffic Trends: Summer spike in "weight loss" queries
```

**Actions to Take**:
- Content performing well? Create similar topics
- Traffic dropping? Check for technical issues or algorithm updates
- Seasonal trends? Prepare content in advance
