## Introduction to Backlink Analytics

### What is SEMrush Backlink Analytics?

SEMrush Backlink Analytics is a comprehensive backlink research and analysis tool that allows you to:
- Analyze your website's backlink profile
- Research competitor backlinks
- Identify link building opportunities
- Monitor backlink quality and toxicity
- Track referring domains and anchor text distribution
- Discover lost and broken backlinks
- Perform competitive backlink gap analysis

### Why Backlink Analytics Matters

Backlinks remain one of the top three ranking factors in Google's algorithm. Understanding your backlink profile helps you:
- Build authority and trust
- Improve search engine rankings
- Identify and remove toxic links
- Find new link building opportunities
- Understand your competitive landscape
- Protect against negative SEO

### Database and Data Freshness

**SEMrush Backlink Database:**
- Over 43 trillion backlinks indexed
- 808 million domains tracked
- Daily updates for monitored projects
- Historical data going back several years
- Coverage across 130+ geographic databases

## Core Metrics Explained

### Authority Score (AS)

**What It Is:**
A compound metric (0-100) measuring domain strength based on:
- Backlink quality and quantity
- Organic search traffic
- Domain age
- Traffic trends

**Real-World Example:**
```
Authority Score Interpretation:

Nike.com: AS 94 - Industry giant, massive authority
Medium.com: AS 93 - High-trust content platform
YourStartup.com: AS 32 - Growing site with moderate authority
NewBlog.com: AS 12 - Fresh site building credibility
```

**How to Use It:**
- Benchmark your site against competitors
- Prioritize link prospects (target domains with AS > 40)
- Track authority improvements over time
- Identify authoritative content opportunities

### Trust Score

**Definition:**
Measures the trustworthiness of a domain based on the quality of its backlink profile.

**Calculation Factors:**
- Links from .gov and .edu domains
- Links from high-trust news sites
- Absence of spammy link patterns
- Natural link velocity

**Real-World Application:**
```
Scenario: E-commerce Health Supplement Store

Your Site Trust Score: 67
Competitor A Trust Score: 82
Competitor B Trust Score: 45

Analysis:
- Competitor A has links from healthline.com, webmd.com (high trust)
- You need medical/health authority backlinks
- Competitor B may be using questionable link tactics
```

### Toxicity Score

**What It Measures:**
Percentage of potentially harmful backlinks that could trigger penalties.

**Toxicity Levels:**
- 0-30%: Healthy profile
- 31-45%: Moderate risk, monitor closely
- 46-60%: High risk, immediate action needed
- 61-100%: Critical, site at penalty risk

**Real-World Example:**
```
Analyzing LocalRestaurant.com:

Toxicity Score: 42%

Toxic Link Examples Found:
1. casino-links-exchange.ru (Spam network)
2. auto-backlink-generator.com (Link scheme)
3. buy-followers-cheap.info (Suspicious domain)

Action Plan:
1. Export toxic links
2. Add to disavow file
3. Submit to Google Search Console
4. Monitor toxicity monthly
```

### Follow vs. NoFollow Ratio

**Healthy Distribution:**
- 70-85% DoFollow links
- 15-30% NoFollow links

**Why It Matters:**
Too many DoFollow links from low-quality sites = Red flag
All DoFollow links = Unnatural link pattern
Balanced mix = Natural, organic growth

**Real-World Benchmark:**
```
TechCrunch.com Analysis:
DoFollow: 73%
NoFollow: 27%
Result: Natural, healthy profile

SpammySite.com Analysis:
DoFollow: 98%
NoFollow: 2%
Result: Suspicious, possible PBN usage
```

### Referring Domains

**Definition:**
Unique domains linking to your site. More important than total backlinks.

**Quality Over Quantity:**
```
Site A: 10,000 backlinks from 50 domains (20 links per domain average)
Site B: 5,000 backlinks from 500 domains (10 links per domain average)

Site B is stronger because:
- More domain diversity
- Natural link pattern
- Broader online presence
- Less reliance on few sources
```

## Anchors Report

### Understanding Anchor Text Distribution

Anchor text is the clickable text in a hyperlink and significantly impacts SEO.

### Anchor Text Types

**1. Exact Match**
- Contains target keyword exactly
- Example: "buy running shoes"

**2. Partial Match**
- Contains keyword variation
- Example: "best running shoes for marathons"

**3. Branded**
- Company or brand name
- Example: "Nike" or "Nike.com"

**4. Naked URL**
- Plain URL as anchor
- Example: "www.nike.com"

**5. Generic**
- Non-descriptive text
- Example: "click here", "this page", "learn more"

**6. Image**
- Image alt text serving as anchor
- Example: [Image of running shoes]

### Healthy Anchor Text Distribution

**Ideal Ratios:**
```
Natural Link Profile:

Branded anchors: 40-50%
- "YourBrand"
- "YourBrand.com"
- "YourBrand Official"

Naked URLs: 15-25%
- "www.yourbrand.com"
- "yourbrand.com/page"

Generic anchors: 10-15%
- "click here"
- "this article"
- "learn more"

Partial match: 10-20%
- "best [category] tool"
- "[keyword] guide"

Exact match: 5-10%
- "[exact keyword]"

LSI/Related: 5-10%
- Related terms
- Synonyms
```
