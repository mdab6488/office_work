
## Getting Started

### Step 1: Account Setup

1. Visit [jetoctopus.com](https://jetoctopus.com)
2. Sign up for a free trial (no credit card required initially)
3. Choose your initial plan based on website size
4. Receive onboarding call (included in all plans)

### Step 2: Create Your First Project

```plaintext
Dashboard → New Project → Enter Website URL
```

**Basic Configuration:**
- **Project Name**: "My E-commerce Site Audit"
- **Start URL**: https://example.com
- **Crawl Limit**: 50,000 pages (adjust based on your plan)
- **Crawl Speed**: Medium (adjust if server can handle more)

### Step 3: Configure Crawl Settings

**Essential Settings:**
- **User Agent**: Googlebot (default)
- **Respect robots.txt**: Yes
- **JavaScript Rendering**: Enable for modern sites
- **Crawl Depth**: Unlimited or set specific depth
- **Follow External Links**: Usually "No"

### Step 4: Start Your First Crawl

Click "Start Crawl" and wait for completion. A 50K page site typically takes 5-10 minutes.
