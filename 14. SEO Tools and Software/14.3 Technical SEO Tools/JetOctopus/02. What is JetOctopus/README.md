
## What is JetOctopus?

JetOctopus is a comprehensive technical SEO platform that combines three critical data sources into one unified interface:

1. **Website Crawler** - Analyzes your site structure, content, and technical elements
2. **Log File Analyzer** - Monitors how search engine bots interact with your website
3. **Data Integration Hub** - Connects GSC, GA4, and other tools for holistic insights

### Primary Use Cases

- Large-scale technical SEO audits
- E-commerce website optimization
- Crawl budget analysis and optimization
- JavaScript SEO troubleshooting
- International and multilingual site audits
- Site migration monitoring
- Content duplication detection
- Internal linking optimization
