
## Pricing & Plans

### Pricing Structure (2025)

JetOctopus uses a flexible, pay-as-you-go pricing model based on pages crawled per month.

**Base Package (Starter):**
```
Price: $237/month (billed annually) or ~$280/month (monthly billing)
Annual billing saves: ~$1,058/year (25% discount)

Includes:
- 500K pages crawled per month
- 250K JavaScript crawl pages
- 2 million log lines (bot traffic only)
- 1 Google Search Console property
- 1 Google Analytics property
- Unlimited projects
- Unlimited users
- Unlimited simultaneous crawls
- 1 onboarding call
- Email support
- Ahrefs integration
- Datasets feature
- Alerts system
- 12-month data retention (if subscription active)
```

**Professional Package:**
```
Price: Starting at $400-600/month (billed annually)

Includes everything in Starter plus:
- 1-2 million pages crawled per month
- 500K-1M JavaScript crawl pages
- 5-10 million log lines
- 3-5 Google Search Console properties
- 3-5 Google Analytics properties
- Priority support
- Custom crawl configurations
- Advanced API access
```

**Enterprise Package:**
```
Price: Custom (typically $1,500-3,000+/month)

Includes everything in Professional plus:
- Up to 100 million pages crawled per month
- Unlimited JavaScript crawling
- Unlimited log lines
- 10+ Google Search Console properties
- 10+ Google Analytics properties
- Dedicated account manager
- Custom integrations
- SLA guarantees
- White-label reporting
- Training sessions
```

**Add-Ons Available:**
```
Extra Log Lines: From $50/month for +5M lines
Extra GSC Properties: $20/month per property
Extra GA4 Properties: $20/month per property
Additional Storage: Custom pricing
```

### Cost Comparison

| Tool | Basic Plan | Features | Best For |
|------|-----------|----------|----------|
| **JetOctopus** | $237/month | 500K pages, unlimited projects, logs analysis | Medium-large sites, agencies |
| **Screaming Frog** | $209/year | Unlimited pages, desktop only, no logs | Small-medium sites, individuals |
| **Ahrefs** | $99/month | Site audit + full SEO suite | All-in-one SEO needs |
| **Botify** | $500+/month | Enterprise crawler, advanced features | Large enterprises |
| **OnCrawl** | $600+/month | Similar to JetOctopus | Large enterprises |
| **DeepCrawl (Lumar)** | $300+/month | Enterprise crawler | Medium-large sites |

**Value Proposition:**
- More affordable than Botify/OnCrawl
- More features than Screaming Frog
- Faster than desktop tools
- No project limits (unlike most competitors)
