
## Pricing and Plans

### Overview of Plans

DeepCrawl/Lumar offers several pricing tiers based on:
- Pages crawled per month
- Number of projects
- Users/seats
- Advanced features
- Support level

**Typical Pricing Structure:**

```yaml
Starter Plan:
  Pages: Up to 100,000/month
  Projects: 3
  Users: 3
  Price: ~$300-500/month
  Best for: Small-medium businesses

Professional Plan:
  Pages: Up to 500,000/month
  Projects: 10
  Users: 10
  Features: + JavaScript rendering, log file analysis
  Price: ~$800-1,200/month
  Best for: Growing e-commerce sites

Enterprise Plan:
  Pages: 1M - 10M+/month
  Projects: Unlimited
  Users: Unlimited
  Features: All features + API access, custom reports
  Price: Custom (typically $2,000-10,000+/month)
  Best for: Large enterprises, agencies
```

### Cost-Benefit Analysis

**ROI Calculation Example:**

```
Scenario: E-commerce site with $500K/month organic revenue

Investment: $1,500/month DeepCrawl Enterprise
Annual Cost: $18,000

Issues Identified and Fixed:
1. 15,000 products noindexed (error) → +$75K/month
2. Page speed optimization → +$30K/month  
3. Internal linking improvements → +$20K/month
4. Duplicate content resolution → +$15K/month

Total Revenue Gain: +$140K/month = +$1.68M/year
ROI: ($1,680,000 - $18,000) / $18,000 = 9,233%

Payback Period: Less than 5 days
```

**Cost vs. Alternatives:**

```
In-House Development:
- Developer time: 40 hours/month @ $100/hr = $4,000/month
- Infrastructure costs: $500/month
- Total: $4,500/month
- Drawbacks: Limited features, maintenance burden

Screaming Frog Desktop:
- License: $250/year per seat
- 5 seats: $1,250/year
- Benefits: One-time per year, familiar tool
- Drawbacks: Manual crawls, no automation, desktop-only, not cloud-based

DeepCrawl/Lumar:
- Cost: $1,500/month = $18,000/year
- Benefits: Automated, scalable, team collaboration, API
- Best for: Enterprise scale and automation needs

Decision:
For < 50,000 pages: Screaming Frog adequate
For 50K - 500K pages: DeepCrawl justified
For 500K+ pages: DeepCrawl essential
```
