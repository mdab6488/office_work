
## Getting Started

### Step 1: Account Setup

1. **Sign Up**: Visit lumar.io and create an account
2. **Choose Plan**: Select plan based on pages crawled per month
3. **Add Users**: Invite team members with role-based access
4. **Configure Settings**: Set up company preferences

### Step 2: Create Your First Project

**Project Setup:**

```
1. Click "Create New Project"
2. Enter Project Details:
   - Project Name: "Main Website"
   - Primary Domain: https://www.example.com
   - Project Type: Website/Mobile App
3. Configure Basic Settings:
   - Timezone
   - Currency
   - Default reports
4. Set Up Integrations:
   - Google Analytics
   - Google Search Console
   - Google PageSpeed Insights
```

### Step 3: Configure Your First Crawl

**Crawl Configuration:**

```yaml
Crawl Name: "Initial Site Audit"
Start URL: https://www.example.com
Crawl Type: Full Site Crawl

Settings:
  Crawl Limit: 50,000 pages
  User Agent: Googlebot Desktop
  JavaScript Rendering: Enabled
  Respect Robots.txt: Yes
  Follow Canonicals: Yes
  Max Crawl Depth: Unlimited
  
Advanced Settings:
  Custom Headers: None
  Cookie Authentication: Disabled
  IP Restriction Bypass: Enabled
  Crawl Rate: Maximum allowed
```

### Step 4: Launch Your Crawl

1. Review all settings
2. Click "Start Crawl"
3. Monitor progress in real-time
4. Receive notification when complete

**Crawl Duration Estimates:**
- 10,000 pages: ~15-30 minutes
- 50,000 pages: ~1-2 hours
- 500,000 pages: ~6-12 hours
- 5,000,000 pages: ~1-2 days
