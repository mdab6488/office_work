
## 1. Introduction to Screaming Frog {#introduction}

### What is Screaming Frog SEO Spider?

Screaming Frog SEO Spider is a **desktop-based website crawler** that acts like a search engine bot, crawling websites to analyze and audit their technical and onsite SEO elements. Think of it as a comprehensive x-ray machine for websites.

### Key Capabilities

- **Website Crawling**: Crawl up to 500 URLs (free) or unlimited (paid)
- **Technical SEO Analysis**: Identify broken links, redirect chains, duplicate content
- **Content Auditing**: Analyze page titles, meta descriptions, headings
- **Site Architecture**: Visualize site structure and internal linking
- **Data Export**: Export findings to Excel, CSV, or Google Sheets
- **Integration**: Connect with Google Analytics, Search Console, PageSpeed Insights

### Who Uses Screaming Frog?

- **SEO Professionals**: Daily technical audits
- **Digital Marketing Agencies**: Client website analysis
- **Web Developers**: Pre-launch site checks
- **Content Strategists**: Content gap analysis
- **E-commerce Managers**: Product page optimization
