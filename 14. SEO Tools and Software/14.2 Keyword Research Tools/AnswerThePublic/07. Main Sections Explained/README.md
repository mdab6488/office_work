
## Main Sections Explained

### Report Types

#### **1. Wheels Report**
**Format:** Circular, color-coded visualization

**Sections:**
- Questions (center radiating outward)
- Prepositions (another wheel)
- Comparisons (another wheel)
- Alphabetical (another wheel)

**Best For:**
- Presentations to clients
- Visual brainstorming sessions
- Identifying content clusters
- Quick pattern recognition

**How to Read:**
- **Center**: Your keyword
- **Inner ring**: Question words (How, What, Why, etc.)
- **Outer ring**: Specific search queries
- **Color intensity**: Popularity (darker = more searches)
- **Hover**: See search volume and CPC

#### **2. Lists Report**
**Format:** Simple bullet-point list

**Best For:**
- Quick scanning
- Copying phrases
- Keyword extraction
- Content ideation sessions

**Features:**
- Alphabetically sorted
- Easy to read
- Fast navigation
- Simple copy-paste

#### **3. Tables Report**
**Format:** Spreadsheet-style table

**Columns:**
- Keyword/Phrase
- Search Volume
- CPC
- Popularity Score
- Category

**Best For:**
- Data analysis
- Comparison
- Filtering by metrics
- Sorting by priority

**Features:**
- Sort by any column
- Filter specific ranges
- Compare multiple metrics
- Export to Excel/Google Sheets

### Filtering Options

**Available Filters:**
1. **Category Filter**: Questions, Prepositions, Comparisons, Alphabetical
2. **Volume Filter**: High, Medium, Low search volume
3. **CPC Filter**: Expensive, Moderate, Cheap keywords
4. **Custom Filter**: Hide unwanted terms (click minus sign)

**Pro Tip:** Click the minus (-) sign next to any phrase to hide it and refine your report.
