
## Getting Started

### Step 1: Access Ubersuggest
1. Visit `ubersuggest.com` or `neilpatel.com/ubersuggest`
2. You can use limited features without signing up
3. Create a free account for more daily searches
4. Consider paid plans for unlimited access

### Step 2: Dashboard Overview
After logging in, you'll see:
- **Search bar**: Enter keywords or domains
- **Left sidebar**: Access different tools
- **Project dashboard**: Manage tracked websites
- **Daily search limit**: Shows remaining searches (free accounts)
