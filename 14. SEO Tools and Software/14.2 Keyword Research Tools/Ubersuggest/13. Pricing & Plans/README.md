
## Pricing & Plans

### Free Plan
- **Cost**: $0/month
- **Limitations**:
  - 3 searches per day
  - Limited data per query
  - Basic features only
- **Best For**: Casual users, testing the tool

### Individual Plan
- **Cost**: ~$12-29/month (varies by promotion)
- **Includes**:
  - 1 website project
  - 150 keywords tracked
  - 1,000 backlink data per month
  - Site audit with daily checks
  - All basic features
- **Best For**: Freelancers, small blogs

### Business Plan
- **Cost**: ~$20-49/month
- **Includes**:
  - 7 website projects
  - 300 keywords tracked
  - 5,000 backlink data per month
  - Priority support
- **Best For**: Agencies, growing businesses

### Enterprise Plan
- **Cost**: ~$40-99/month
- **Includes**:
  - 15 website projects
  - 900 keywords tracked
  - 10,000 backlink data per month
  - White-label reports
  - API access
- **Best For**: Large agencies, enterprises

### Lifetime Deal
Occasionally available:
- One-time payment ($290-$990)
- Lifetime access to features
- Worth it if you're committed long-term
