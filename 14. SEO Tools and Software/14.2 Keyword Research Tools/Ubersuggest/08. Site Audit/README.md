
## Site Audit

### Running a Comprehensive Site Audit

#### Step 1: Create a Project
1. Click "Add Project" in dashboard
2. Enter your website URL
3. Set tracking preferences

#### Step 2: Review SEO Score
- Overall SEO Health Score (0-100)
- Critical Errors (must fix immediately)
- Warnings (should fix soon)
- Recommendations (nice to have)

### Audit Categories

#### 1. **On-Page SEO**
- Missing title tags
- Duplicate meta descriptions
- Missing H1 tags
- Thin content (< 300 words)
- Missing alt text on images

#### 2. **Site Speed**
- Page load time
- Server response time
- Large image files
- Render-blocking resources

#### 3. **Mobile Optimization**
- Mobile-friendliness
- Responsive design issues
- Viewport configuration

#### 4. **Crawlability**
- Broken links (404 errors)
- Redirect chains
- Blocked resources
- Robots.txt issues
- XML sitemap problems

### Real-Life Example 6: E-commerce Site Audit

**Scenario**: Online store with declining traffic.

**Process**:
1. Run site audit on "mystore.com"
2. Results:
   - SEO Score: 62/100
   - Critical Errors: 23
   - Warnings: 47
   - Recommendations: 112

3. Key findings:
   - **15 broken links** on product pages
   - **45 product pages** missing meta descriptions
   - **Site speed**: 4.2 seconds (should be < 3s)
   - **12 pages** with duplicate title tags

**Action Plan**:
1. **Week 1**: Fix all broken links (traffic drop might be from 404s)
2. **Week 2**: Write unique meta descriptions for top 20 product pages
3. **Week 3**: Optimize images (compressed, resized)
4. **Week 4**: Fix duplicate titles, create unique ones

**Results After 8 Weeks**:
- SEO Score improved to 87/100
- Organic traffic increased by 34%
- Bounce rate decreased from 68% to 52%
