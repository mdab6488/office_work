#### Critical Issues - Deep Dive

**Issue 1: 404 Errors (43 pages)**

**Detailed Report:**

```
Page                                  | Links To | Source Pages
--------------------------------------|----------|------------------
/products/organic-apples-old          | 12       | Category, Search
/blog/summer-recipes-2023             | 8        | Blog archive
/seasonal/thanksgiving-deals          | 15       | Homepage, Email
/category/gluten-free-old             | 23       | Main menu

Root Causes:
- Products discontinued without redirects
- Old blog posts deleted
- Seasonal pages removed after campaign
- Category restructuring without URL mapping

Impact:
- Lost link equity from 58 internal links
- Poor user experience
- Wasted crawl budget
- Negative SEO signals
```

**Solution in AWR:**

```
Export 404 Report:
Reports → Website Audit → 404 Errors → Export CSV

Action Plan:
1. Immediate Fixes (High Traffic URLs):
   /products/organic-apples-old → 301 redirect to /products/organic-apples
   /seasonal/thanksgiving-deals → 301 redirect to /seasonal-offers

2. Blog Content (Outdated):
   /blog/summer-recipes-2023 → 301 redirect to /blog/summer-recipes
   OR update content to 2025 version

3. Category Changes:
   /category/gluten-free-old → 301 redirect to /category/gluten-free

4. Update Internal Links:
   - Replace all instances pointing to old URLs
   - Update sitemap
   - Notify Google via Search Console
```

**Issue 2: Duplicate Content (12 pages)**

**Examples Found:**

```
Duplicate Set 1:
Primary URL: /products/organic-milk
Duplicates:
- /products/organic-milk?sort=price (URL parameters)
- /products/organic-milk?ref=homepage (tracking parameter)
- /products/organic-milk/print-view (print version)

Duplicate Set 2:
Primary URL: /category/dairy
Duplicates:
- /category/dairy/page-1 (pagination)
- /category/dairy/?filter=all (filter parameter)
```

**AWR Analysis:**

```
Duplicate Content Report:
- 12 pages identified
- 4 duplicate sets
- Content similarity: 95-100%

SEO Impact:
- Diluted ranking signals
- Confused search engines on which page to rank
- Wasted crawl budget
- Potential ranking penalties
```

**Solution Strategy:**

```
Fix 1: Canonical Tags
Add to all product pages:
<link rel="canonical" href="https://www.freshgrocery.com/products/organic-milk" />

Implementation:
<!-- All variants should point to clean URL -->
/products/organic-milk?sort=price → canonical to /products/organic-milk
/products/organic-milk?ref=homepage → canonical to /products/organic-milk

Fix 2: URL Parameters in Search Console
Configure Google Search Console:
- Parameter: sort → Doesn't change content → Let Googlebot decide
- Parameter: ref → Doesn't change content → No URLs
- Parameter: filter → Changes content → Let Googlebot crawl

Fix 3: Robots.txt for Print Versions
Disallow: /*/print-view/

Fix 4: Pagination Tags
<link rel="prev" href="/category/dairy" />
<link rel="next" href="/category/dairy/page-2" />
```

**Optimization Action Plan:**

```
  - CSS/JS: cache for 1 month

□ Combine CSS files

□ Code splitting for JavaScript
```

```python
# Get GSC data
request = {
    'startDate': '2025-10-10',
    'endDate': '2025-11-09',
    'dimensions': ['query'],
    'rowLimit': 1000
}

    siteUrl='https://downtowndental.com/',
    body=request
).execute()

# Convert to DataFrame
gsc_data = pd.DataFrame(response['rows'])
gsc_data['query'] = gsc_data['keys'].apply(lambda x: x[0])

    right_on='query',
```

```
Benefits of Focused Approach:
- Clearer competitive insights
- Actionable intelligence
- Less data overwhelm
- Focused strategy development
```

## Common Issues & Solutions

### Issue 1: Rankings Don't Match Manual Checks

**Problem:**
AWR shows position #5, but when you Google it, you see #3.

**Causes & Solutions:**

```
Cause 1: Personalization
Your manual search is personalized based on:
- Search history
- Location
- Logged-in Google account
- Previous site visits

Solution:
Use Incognito/Private mode for manual checks
Or use AWR's SERP view (shows actual rankings)

Cause 2: Location Differences
AWR checks from specific location you set
Your actual location might be different

Solution:
Verify location settings in AWR match your test location
Use VPN to match AWR's check location

Cause 3: Device Type
AWR checked mobile, you checked desktop (or vice versa)

Solution:
Check which device AWR is tracking
Make sure you're comparing apples to apples

Cause 4: Timing
Rankings fluctuate throughout the day
AWR checked at 3 AM, you checked at 2 PM

Solution:
Check AWR's update time
Run manual update and check immediately after

Cause 5: SERP Features
Featured snippet or local pack affecting what you see

Solution:
AWR distinguishes between organic, local pack, and features
Check specific result type

Best Practice:
Trust AWR over manual checks - it's unbiased and consistent
Manual checks are useful for SERP analysis, not accurate ranking
```
