## Introduction

Frase is an AI-powered content optimization platform designed to help content creators, SEO professionals, and marketers research, write, and optimize content that ranks higher in search engines. It combines SERP analysis, content briefs, AI writing, and optimization scoring into a single workflow.

**Primary Purpose**: Bridge the gap between content research and creation by analyzing top-ranking content and providing data-driven recommendations.

---

## What is Frase?

### Core Capabilities

Frase is a **content intelligence platform** that:
- Analyzes top 20 Google search results for any keyword
- Creates research-driven content briefs
- Provides AI-assisted writing tools
- Scores content against competitors
- Identifies semantic keywords and topics
- Generates content outlines automatically

### Who Should Use Frase?

- **Content Writers**: Create well-researched, SEO-optimized content faster
- **SEO Specialists**: Develop comprehensive content strategies
- **Content Marketers**: Scale content production with quality
- **Agencies**: Manage multiple client content projects
- **Bloggers**: Compete with established websites

## Getting Started

### Initial Setup

**Step 1: Create an Account**
```
1. Visit frase.io
2. Sign up for a free trial or paid plan
3. Complete onboarding tutorial
4. Connect Google Search Console (optional)
```

**Step 2: Create Your First Document**
```
1. Click "New Document"
2. Enter target keyword (e.g., "best running shoes for beginners")
3. Select search location (United States, United Kingdom, etc.)
4. Choose language
5. Click "Create Document"
```

**Step 3: Wait for SERP Analysis**
- Frase crawls and analyzes top 20 results (takes 30-60 seconds)

## Content Brief Creation

### Understanding Content Briefs

A **content brief** is a research document that includes:
- Keyword analysis
- Competitor insights
- Suggested headings
- Questions to answer
- Key topics to cover

### Real-World Example: "How to Start a Podcast"

**Step 1: Enter Keyword**
```
Target Keyword: "how to start a podcast"
Location: United States
Language: English
```

**Step 2: Review SERP Results**

```

Average Word Count: 3,250 words
```

## Best Practices

### 1. Don't Chase Perfect Scores

**❌ Wrong Approach**:
```
"I need 100/100 score before publishing"
Result: Over-optimized, keyword-stuffed content
```

**✅ Right Approach**:
```
"I'll aim for 80-90/100 with natural writing"
Result: Well-optimized, readable content
```

**Why**: Scores above 90 often require keyword stuffing that hurts readability. Google prioritizes user experience over keyword density.

### 2. Use AI Writing Sparingly

**❌ Wrong**:
```
Generate entire article with AI tools
Copy/paste without editing
```

**✅ Right**:
```
Use AI for:
- Overcoming writer's block
- Generating outlines
- Creating FAQ answers
- Rewriting awkward sentences

Always: Add human insight, examples, and personality
```

### 3. Prioritize Topic Coverage Over Word Count

**❌ Wrong Focus**:
```
"Competitors have 4,000 words, so I need 4,500"
```

**✅ Right Focus**:
```
"Competitors cover these 12 topics thoroughly. I'll cover all 12 plus add 2 unique angles"
```

**Example**:
```
Competitor Article (4,000 words):
- Covers 12 topics superficially
- Some fluff and repetition

Your Article (3,200 words):
- Covers same 12 topics with depth
- Adds 2 unique expert insights
- No fluff

Result: Better rankings despite fewer words
```

### 4. Update SERP Analysis Quarterly

**Why**: Search results change over time

**Process**:
```
Every 3 Months:
1. Re-run SERP analysis in Frase
2. Check for new competitors
3. Identify new topics being covered
4. Update your content accordingly
5. Re-optimize score
```

### 5. Combine Frase with Original Research

**Frase Shows**: What competitors are saying
**You Add**: What competitors are missing

**Example**:
```
Topic: "Email Marketing ROI"

Frase Analysis shows competitors mention:
- Average ROI is $36 per $1 spent
- Email has highest ROI of any channel
- Segmentation improves results

You Add (original research):
- Survey of 500 businesses in your industry
- Specific ROI benchmarks by company size
- Case study from your client
- Expert interview with email marketing leader

Result: More authoritative, citable content
```

### 6. Don't Ignore User Intent

**Warning**: Frase shows what ranks, not always what users want

**Example**:
```
Search Query: "best CRM for small business"

Frase Shows:
- Long comparison articles rank
- Affiliate review sites dominate

But Consider:
- User might want quick answer
- Interactive tool could be better
- Video comparison could work

Solution: Use Frase data + user empathy
```

## Common Mistakes to Avoid

### Mistake 1: Keyword Stuffing for Higher Scores

**Problem**: Forcing keywords unnaturally to reach 90+ scores

**Example of Over-Optimization**:
```
❌ Bad: "The best coffee makers are important because coffee makers help you make coffee. When choosing coffee makers, consider which coffee makers have the features you need in coffee makers."

✅ Good: "When choosing a coffee maker, consider the features that matter most for your brewing style. High-quality machines offer programmability, temperature control, and consistent extraction."
```

**Solution**: Stop at 80-88 score if writing naturally

### Mistake 2: Copying Competitor Structure Exactly

**Problem**: Creating duplicate content with same structure

**Example**:
```
Competitor 1: Introduction → Benefits → How-To → FAQs → Conclusion
Competitor 2: Introduction → Benefits → How-To → FAQs → Conclusion  
Your Article: Introduction → Benefits → How-To → FAQs → Conclusion

Result: Google sees little differentiation
```

**Solution**: Use Frase for topics, but create unique structure and angles

### Mistake 3: Ignoring Search Intent

**Example**:
```
Keyword: "iPhone 15 review"
Search Intent: People want detailed product review

Your Article Based Solely on Frase:
- 80% buying guide content
- 10% comparison
- 10% actual review

Result: High score, but wrong content for intent
Rankings: Poor despite optimization
```

**Solution**: Always validate Frase insights against actual search intent

### Mistake 4: Not Updating Content

**Problem**: Creating content once and never refreshing

**Reality**:
```
Month 1: Position 5 (score: 87/100)
Month 6: Position 9 (score: 87/100) - Why?

Reason: Competitors updated their content
Their new scores: 90-95/100
Your stale content: Still 87/100
```

**Solution**: Re-analyze in Frase every 3-6 months

### Mistake 5: Relying Only on Frase

**Incomplete Workflow**:
```
❌ Frase → Write → Publish

✅ Keyword Research → Frase → Write → Edit → Optimize On-Page SEO → Get Backlinks → Publish → Promote → Monitor
```

**Remember**: Frase optimizes content, but doesn't:
- Build backlinks
- Improve technical SEO
- Increase page speed
- Drive initial traffic

### Mistake 6: Forgetting About E-E-A-T

**Google's E-E-A-T**: Experience, Expertise, Authoritativeness, Trust

**What Frase Can't Do**:
```
❌ Add author credentials
❌ Include original research
❌ Build trust signals
❌ Show real experience
```

**What You Must Add Manually**:
```
✅ Author bio with credentials
✅ Personal experience and examples
✅ Original data and insights
✅ Citations to authoritative sources
✅ Updated dates
✅ Expert quotes
```

### Mistake 7: Neglecting Readability

**Problem**: Optimizing for Frase score but not humans

**Example**:
```
Frase Score: 92/100
Readability: Poor
- Paragraphs: 8-10 sentences each
- Sentences: 30-40 words
- No subheadings in long sections
- No images breaking up text

Result: High bounce rate despite good rankings
```

**Solution**: Balance Frase score with readability metrics
- Paragraph length: 2-4 sentences
- Sentence length: 15-20 words average
- Subheading every 300 words
- Images every 400-500 words
