## 3. On-Page SEO

On-Page SEO refers to all optimization techniques applied directly to your website's pages to improve rankings and user experience. Unlike off-page SEO (backlinks, social signals), you have complete control over on-page elements.

---

## 3.1 Content Optimization



### Content Formatting

**What It Is:**
How you structure and present your content visually to improve readability and user experience.

**Why It Matters:**
78% of users scan rather than read web content (Nielsen Norman Group study). Proper formatting helps users find information quickly, reducing bounce rates and improving engagement metrics.

**Formatting Best Practices:**
- Use H2/H3 headers every 300-500 words
- Keep paragraphs to 2-4 sentences max
- Add white space between sections
- Use bullet points for lists
- Include relevant images every 300-500 words
- Use bold for key terms (sparingly)
- Add tables for comparisons
- Include emojis occasionally for visual breaks (use sparingly)
- Use blockquotes for important callouts
