# Complete Guide to Header Tags & URL Optimization

## 3.6 URL Optimization

URLs are a critical ranking factor and user experience element. A well-optimized URL tells both users and search engines exactly what the page is about.

## Summary Checklist

### Header Tags Checklist:
- [ ] One H1 tag per page with target keyword
- [ ] H1 is descriptive and under 70 characters
- [ ] H2-H6 follow logical hierarchy (no skipping levels)
- [ ] Headers create a clear content outline
- [ ] Keywords included naturally in subheadings
- [ ] Headers are descriptive, not generic
- [ ] No keyword stuffing in headers
- [ ] Headers improve content scannability

### URL Optimization Checklist:
- [ ] URLs are descriptive and readable
- [ ] Primary keyword included in URL
- [ ] URLs use hyphens (-) not underscores (_)
- [ ] All URLs are lowercase
- [ ] URL length under 100 characters ideally
- [ ] Unnecessary words removed
- [ ] Logical category structure
- [ ] No special characters or spaces
- [ ] No keyword stuffing in URL
- [ ] Old URLs redirected if changed (301 redirects)

---

## Quick Reference Guide

### Header Tag Formula:
```
H1: [Target Keyword] + [Modifier] + [Value Proposition]
H2: [Main Topic Aspect] + [Supporting Keywords]
H3: [Specific Details] + [Natural Language]
```

### URL Formula:
```
https://domain.com/[primary-keyword]-[modifier]-[location/category]

Keep it: Short + Descriptive + Keyword-Rich + Lowercase + Hyphenated
```

## Additional Resources

### For Header Tags:
- Test your header structure with browser extensions like "HeadingsMap"
- Validate hierarchy with W3C Markup Validator
- Check competitor header strategies with "SEO Meta in 1 Click"

### For URLs:
- Use Google Search Console to identify URL issues
- Implement 301 redirects for any URL changes
- Test case sensitivity on your server environment
- Monitor URL performance in Google Analytics

---

**Remember:** Both header tags and URLs are foundational elements of on-page SEO. They work together to signal content relevance to search engines while improving user experience. Always prioritize clarity and user intent over SEO tricks.
