## 3.7 Image SEO

Image SEO is crucial for driving organic traffic from Google Images, improving page load speed, and enhancing user experience. Properly optimized images can account for 20-60% of a website's total traffic.


### Image File Names

**What It Is:**
The filename of your image is one of the first signals Google uses to understand what the image contains.

**Why It Matters:**
- Google reads file names to understand image context
- Descriptive names improve image search rankings
- Helps accessibility tools describe images
- Improves organization and management

**Best Practices:**

✅ **DO:**
```
nike-air-max-90-white-sneakers.jpg
chocolate-chip-cookie-recipe-2024.jpg
golden-retriever-puppy-playing-park.jpg
macbook-pro-16-inch-2024-silver.jpg
```

❌ **DON'T:**
```
IMG_12345.jpg
DSC00987.jpg
image1.jpg
photo.jpg
untitled.png
```

**Real-Life Example:**

**E-commerce Store (Nike):**
```
Bad:  product-123456.jpg
Good: nike-air-jordan-1-retro-high-chicago-red-black-white.jpg
```

**Recipe Blog:**
```
Bad:  recipe_final_v2.jpg
Good: homemade-pepperoni-pizza-recipe-step-by-step.jpg
```

**Photography Portfolio:**
```
Bad:  _MG_4532.jpg
Good: sunset-golden-gate-bridge-san-francisco-2024.jpg
```

**Rules to Follow:**
- Use hyphens (-) not underscores (_)
- Keep it descriptive but under 5 words when possible
- Include primary keywords naturally
- Use lowercase letters
- Avoid special characters and spaces
