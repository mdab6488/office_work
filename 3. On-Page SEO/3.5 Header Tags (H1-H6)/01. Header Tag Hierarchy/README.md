# Complete Guide to Header Tags & URL Optimization

## 3.5 Header Tags (H1-H6)

Header tags are HTML elements used to define headings and subheadings on a webpage. They serve dual purposes: organizing content for readers and signaling content hierarchy to search engines.



### Header Tag Hierarchy

Header tags follow a hierarchical structure from H1 (most important) to H6 (least important). Think of them as an outline for your content.

#### The Hierarchy Explained:

```
H1: Main Title/Topic
├── H2: Major Section
│   ├── H3: Subsection
│   │   ├── H4: Sub-subsection
│   │   │   ├── H5: Minor point
│   │   │   │   └── H6: Very specific detail
```

#### Real-Life Example - Recipe Blog Post:

**Poor Structure:**
```html
<h1>Chocolate Chip Cookies</h1>
<h3>Ingredients</h3>
<h2>Baking Instructions</h2>
<h4>Preparation</h4>
<h2>Tips</h2>
```

**Proper Structure:**
```html
<h1>The Ultimate Chocolate Chip Cookie Recipe</h1>

<h2>Ingredients You'll Need</h2>
<h3>Dry Ingredients</h3>
<h3>Wet Ingredients</h3>

<h2>Step-by-Step Baking Instructions</h2>
<h3>1. Preparation</h3>
<h4>Mixing the Dough</h4>
<h4>Chilling Time</h4>

<h3>2. Baking Process</h3>
<h4>Oven Temperature Settings</h4>
<h4>Baking Duration</h4>

<h2>Pro Tips for Perfect Cookies</h2>
<h3>Troubleshooting Common Issues</h3>
<h3>Storage Recommendations</h3>
```

**Why This Matters:**
- Search engines understand your content structure better
- Screen readers can navigate more easily (accessibility)
- Users can scan and find information quickly
- Improves on-page SEO signals
