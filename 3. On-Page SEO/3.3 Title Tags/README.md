## 3.3 Title Tags

### What is a Title Tag?

A title tag is an HTML element that specifies the title of a web page. It appears in three critical places:

1. **Search Engine Results Pages (SERPs)** - As the clickable headline
2. **Browser Tabs** - At the top of your browser
3. **Social Media** - When pages are shared (if no Open Graph tags)

**HTML Syntax:**
```html
<head>
  <title>Your Page Title Goes Here</title>
</head>
```
