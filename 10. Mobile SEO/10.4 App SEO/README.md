## 10.4 App SEO

App SEO ensures your mobile app is discoverable in both app stores and search engines.

## Best Practices Summary

### Mobile UX Checklist
- ✅ Sticky navigation with essential actions visible
- ✅ Touch targets minimum 44x44px (iOS) or 48x48px (Android)
- ✅ Forms use correct input types
- ✅ No intrusive interstitials blocking main content
- ✅ Single-column form layout
- ✅ Real-time validation with clear error messages

### ASO Checklist
- ✅ Keyword-optimized title (30 chars iOS, 50 chars Android)
- ✅ Compelling subtitle/short description
- ✅ First 3 screenshots show main value
- ✅ 15-30 second preview video
- ✅ Localized for top markets
- ✅ Regular A/B testing of store assets
- ✅ Prompt for reviews at positive moments
- ✅ Respond to all reviews

### Deep Linking Checklist
- ✅ Universal Links (iOS) configured
- ✅ App Links (Android) verified
- ✅ apple-app-site-association file published
- ✅ assetlinks.json file published
- ✅ Deferred deep linking for installs
- ✅ Smart banners on mobile web
- ✅ Fallback URLs for users without app

### App Indexing Checklist
- ✅ Firebase App Indexing implemented (Android)
- ✅ CoreSpotlight indexing (iOS on-device)
- ✅ NSUserActivity for cloud indexing (iOS)
- ✅ Matching web pages for all app content
- ✅ App verified in Google Search Console
- ✅ Structured data on web pages
- ✅ Log user actions for ranking signals

## Common Pitfalls to Avoid

### Mobile UX
❌ Forcing desktop navigation patterns on mobile
❌ Forms with too many required fields
❌ Popup that appears on every page visit
❌ Small text (under 16px causes iOS zoom)
❌ Non-responsive tables

### ASO
❌ Keyword stuffing in title
❌ Screenshots without context or captions
❌ Ignoring negative reviews
❌ Never updating store listing
❌ Using same assets across all countries

### Deep Linking
❌ Not handling link failures gracefully
❌ Forgetting to update associated domains
❌ No fallback for users without app
❌ Breaking deep links with app updates
❌ Not testing across iOS/Android versions

### App Indexing
❌ App content without web equivalent
❌ Indexing temporary or user-specific content
❌ Not removing deleted content from index
❌ Mismatched URLs between app and web
❌ Not verifying domain ownership
