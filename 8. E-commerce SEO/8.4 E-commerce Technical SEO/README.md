# E-commerce SEO: Advanced Guide

## 8.4 E-commerce Technical SEO

Technical SEO ensures search engines can crawl, index, and understand your e-commerce site efficiently. Poor technical SEO can waste crawl budget and prevent your pages from ranking.

## Summary Checklist

### Category Page Optimization
- [ ] Write unique 300-600 word category descriptions
- [ ] Include both above-fold and below-fold content
- [ ] Implement strategic internal linking (50-100 links)
- [ ] Use descriptive anchor text
- [ ] Add breadcrumb navigation with Schema markup
- [ ] Optimize title tags and meta descriptions
- [ ] Include category Schema markup

### Technical SEO
- [ ] Keep temporarily out-of-stock products live (HTTP 200)
- [ ] Redirect permanently discontinued products (301)
- [ ] Maintain seasonal product pages year-round
- [ ] Choose appropriate pagination strategy
- [ ] Implement crawlable pagination links
- [ ] Use single URLs for simple variants
- [ ] Set up proper canonical tags
- [ ] Test canonicals in Search Console

## Practice Exercise

**Scenario:** You run an online store selling outdoor gear. Your "Camping Tents" category has:
- 150 products
- Products in multiple subcategories (backpacking, family, 4-season)
- Some seasonal products (summer tents)
- Products with variants (color, capacity)
- Several out-of-stock items

**Task:** Create an implementation plan covering:
1. Category description strategy
2. Internal linking structure
3. Breadcrumb implementation
4. Seasonal product handling
5. Variant URL strategy
6. Canonical tag placement
