
### 7.2.3 Citation Audit

#### What is a Citation Audit?

A systematic review of all your existing citations to:
- Find inconsistencies
- Discover duplicate listings
- Identify missing citations
- Find incorrect information

**When to Audit:**
- Before building new citations
- After moving or rebranding
- If rankings suddenly drop
- Annually for maintenance

#### How to Conduct a Citation Audit

**Step 1: Discover Existing Citations**

**Manual Method:**

Google these search queries:

```
"Business Name"
"Business Name" + "City"
"Business Name" + "Phone Number"
"Phone Number"
"Address"
```

**Step 3: Identify Issues and Prioritize**

**Issue Categories:**

**Critical (Fix Immediately):**
- Incorrect address
- Incorrect phone number
- Duplicate listings
- Wrong business name
- Inconsistent NAP on high-authority sites (Google, Yelp, BBB)

**High Priority (Fix This Month):**
- Incomplete profiles on major sites
- Outdated hours
- Missing categories
- Unclaimed listings

**Medium Priority (Fix This Quarter):**
- Formatting inconsistencies
- Missing photos
- Incomplete descriptions
- Low-value outdated listings

**Low Priority (Monitor):**
- Very old mentions/links (2+ years)
- Sites with minimal traffic
- Blogs/articles (unstructured citations)

**Step 4: Fix Citations Systematically**

**Fixing Process:**

**For Claimed Listings:**
1. Log in to platform
2. Update information directly
3. Save changes
4. Verify changes appear
5. Mark as complete in spreadsheet

**For Unclaimed Listings:**
1. Claim the listing
2. Verify ownership (varies by platform)
3. Update all information
4. Complete profile fully
5. Mark as complete

**For Listings You Can't Edit:**
1. Contact platform support
2. Provide proof of ownership
3. Request update or removal
4. Follow up weekly
5. Document attempts


- Top 20 citations manually
- Google search for NAP variations

**Annual Deep Audit:**
- Full citation scan
- Check for new directories
- Remove old/dead citations
- Update for any business changes
