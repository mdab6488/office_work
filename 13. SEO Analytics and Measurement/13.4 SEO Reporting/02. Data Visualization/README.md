**6. Data Visualization Mistakes to Avoid**

**âŒ Don't:**
```
- Use 3D charts (distorts data perception)
- Overcrowd with too many data points
- Use too many colors (stick to 3-5)
- Manipulate axis scales to exaggerate trends
- Show data without context or benchmarks
- Use pie charts with 10+ slices
- Forget to label axes and add units
```

**âœ… Do:**
```
- Keep it simple and focused
- Use consistent colors across reports
- Add clear titles and labels
- Include data sources and date ranges
- Highlight key insights
- Make it actionable
- Test with your audience first
```
