
### 2.2 Performance Reports

#### Understanding Performance Metrics

**Key Metrics Explained:**

```plaintext
┌─────────────────────────────────────────────────────┐
│ Total Clicks                                        │
│ Number of times users clicked your site in Search  │
│ Example: 15,234 clicks last month                  │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ Total Impressions                                   │
│ How many times your site appeared in search         │
│ Example: 450,000 impressions last month            │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ Average CTR (Click-Through Rate)                    │
│ Clicks ÷ Impressions × 100                         │
│ Example: 15,234 ÷ 450,000 = 3.39%                 │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ Average Position                                     │
│ Average ranking in search results                    │
│ Example: Position 8.5 (usually page 1)             │
└─────────────────────────────────────────────────────┘
```

**What Good Metrics Look Like:**

| Metric | Poor | Average | Good | Excellent |
|--------|------|---------|------|-----------|
| CTR | <2% | 2-5% | 5-10% | >10% |
| Average Position | >20 | 11-20 | 4-10 | 1-3 |
| Impressions | Declining | Stable | Growing | Rapid Growth |
