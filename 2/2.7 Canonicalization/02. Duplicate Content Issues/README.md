## 2.7 Canonicalization


### Duplicate Content Issues

#### What Causes Duplicate Content?

**1. HTTP vs HTTPS**
```
http://example.com
https://example.com
```

**2. WWW vs Non-WWW**
```
https://example.com
https://www.example.com
```

**3. Trailing Slashes**
```
https://example.com/products
https://example.com/products/
```
