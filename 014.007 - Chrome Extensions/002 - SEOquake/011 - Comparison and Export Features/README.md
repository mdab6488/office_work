
## Comparison and Export Features

### URL/Domain Comparison

**Feature**: Compare up to 5 URLs or domains side-by-side

### Real-World Comparison Example

**Scenario**: Comparing competitor product pages

**Step 1: Select URLs**
```
Your product: yoursite.com/premium-coffee
Competitor 1: competitor1.com/coffee-beans
Competitor 2: competitor2.com/organic-coffee
```

**Step 2: Run Comparison**
```
Click SEOquake > "Compare URLs/Domains"
Enter URLs
Click "Compare"
```

### Export and Analysis

**Export Formats**:
- CSV (most common)
- Excel compatibility
- Google Sheets import

**Real-World Export Workflow**:

**Scenario**: Monthly competitive analysis

```
Step 1: Export SERP Data
Search: "project management software"
Click SEOquake icon > Export
Save as: serp-project-mgmt-jan-2024.csv

Step 2: Import to Spreadsheet
Open in Google Sheets or Excel

Step 3: Analysis Template

Column Setup:
A: Rank Position
B: URL
C: Title
D: Google Index
E: Backlinks
F: Domain Age
G: SEMrush Rank
H: Facebook Shares
I: Notes

Step 4: Calculate Averages
- Average backlinks: =AVERAGE(E2:E11)
- Average domain age: =AVERAGE(F2:F11)
- Your position vs average

Step 5: Track Over Time
Create new sheet for each month
Track changes:
- Position movements
- Metric changes
- New competitors
- Lost competitors

Step 6: Visualization
Create charts:
- Backlink distribution
- Age vs rank correlation
- Your progress over time
- Competitor movements
```
