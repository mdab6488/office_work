**Accessing Settings**:
```
Click SEOquake icon > Settings (gear icon)
```

### Parameter Customization

**Available Parameters**:
Over 30 metrics can be shown/hidden

**Default Parameters (Most Useful)**:
```
✅ Google Index
✅ SEMrush Rank
✅ SEMrush Backlinks
✅ SEMrush Traffic
✅ Domain Age
✅ Facebook Likes
✅ Last Update (cache)
```

**Custom Configuration Examples**:

**Example 1: Local SEO Focus**
```
Enabled Parameters:
✅ Google Index
✅ Bing Index
✅ Domain Age
✅ Facebook Shares
✅ Google Reviews (if available)
✅ Local Citations
❌ Alexa Rank (not relevant)
❌ Twitter (less important for local)
```

**Example 2: Link Building Focus**
```
Enabled Parameters:
✅ SEMrush Backlinks
✅ Total Backlinks
✅ Referring Domains
✅ DoFollow Links
✅ NoFollow Links
✅ Domain Authority (if available)
✅ Page Authority (if available)
❌ Social metrics (not priority)
```

**Example 3: Content Marketing Focus**
```
Enabled Parameters:
✅ Google Index
✅ SEMrush Traffic
✅ Facebook Shares
✅ Twitter Shares
✅ Pinterest Pins
✅ LinkedIn Shares
✅ Last Updated
❌ Technical metrics (less relevant)
```

### SERP Overlay Customization

**Configure What Shows in Search Results**:

```
Your Typical Workflow: Keyword Research

Minimal SERP Overlay:
✅ Rank Position
✅ SEMrush Rank (traffic indicator)
✅ Backlinks (authority indicator)
✅ Domain Age (trust indicator)
❌ Social metrics (clutters view)
❌ Technical details (not needed in SERP)

Result: Clean, fast overview of competition
```
