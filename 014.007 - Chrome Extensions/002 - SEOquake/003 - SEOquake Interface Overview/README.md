
## SEOquake Interface Overview

### Main Components

#### 1. **SEOquake Toolbar (SEO Bar)**
Located at the top of any webpage when enabled:
```
[Google Index] [SEMrush Rank] [Alexa Rank] [Quantcast] [Links] [Age] [More...]
```

#### 2. **SERP Overlay**
Appears on search engine results pages showing metrics for each result:
```
Search Result Title
URL
Snippet Text
[Rank | Google Index | Links | Domain Age | Social Signals]
```

#### 3. **Drop-Down Menu**
Click SEOquake icon to access:
- Diagnosis (On-Page Audit)
- Density (Keyword Analysis)
- Internal Links
- External Links
- Compare URLs/Domains
- Settings

#### 4. **Side Panel (When Opened)**
Detailed information panel showing:
- Complete parameter list
- Expandable sections
- Export options
- Quick actions
