
## Installation & Setup

### Step 1: Install the Extension

#### For Google Chrome:

```
1. Open Chrome browser
2. Navigate to: chrome.google.com/webstore
3. Search for "Keywords Everywhere"
4. Click on the official extension (verify publisher: Axeman Tech)
5. Click "Add to Chrome" button
6. In popup, click "Add Extension" to confirm
7. Extension icon appears in toolbar (puzzle piece icon)
8. Pin extension for easy access (right-click icon ‚Üí Pin)
```

#### For Mozilla Firefox:

```
1. Open Firefox browser
2. Navigate to: addons.mozilla.org
3. Search for "Keywords Everywhere"
4. Click on the official add-on
5. Click "Add to Firefox"
6. Click "Add" in permission popup
7. Extension icon appears in toolbar
```

### Step 2: Create an Account

```
1. Click the Keywords Everywhere icon in toolbar
2. Click "Register" or "Sign Up"
3. Enter your email address
4. Create a password (min 8 characters)
5. Check your email for verification link
6. Click verification link
7. Return to browser and click extension icon
8. Click "Login"
9. Enter credentials
10. Extension activates (green checkmark appears)
```

**Account Dashboard Access:**
- URL: keywordseverywhere.com/account
- Manage credits
- View usage history
- Update settings
- Purchase credit packages

### Step 4: Purchase Credits (For Paid Features)

**Free Version Limitations:**
- Related keywords: 10 per search
- No search volume data
- No CPC data
- No competition metrics
- Limited trend data

**To Activate Paid Features:**

```
1. Click extension icon
2. Click "Buy Credits" button
3. Choose package:
   - $10 = 100,000 credits
   - $50 = 500,000 credits
   - $100 = 1,000,000 credits
   - $200 = 2,000,000 credits
4. Select payment method (Card/PayPal)
5. Complete purchase
6. Credits activate automatically
7. Check extension icon for credit balance
```

**Credit Consumption:**
- 1 keyword search = 1 credit
- Bulk upload keywords = 1 credit each
- Related keywords = included (no extra cost)
- Exporting data = no credits used
