
## Title Tag Best Practices

### 1. Make Every Title Unique

**❌ Bad Example (Duplicate Titles):**
```html
<!-- Homepage -->
<title>Best Coffee Shop</title>

<!-- Menu Page -->
<title>Best Coffee Shop</title>

<!-- Contact Page -->
<title>Best Coffee Shop</title>
```

**✅ Good Example (Unique Titles):**
```html
<!-- Homepage -->
<title>Best Coffee Shop in Brooklyn - Artisan Coffee & Pastries | Blue Bean Café</title>

<!-- Menu Page -->
<title>Coffee Menu - Espresso, Lattes & Specialty Drinks | Blue Bean Café</title>

<!-- Contact Page -->
<title>Contact Us - Visit Our Brooklyn Location | Blue Bean Café</title>
```

**Why it matters:** Duplicate titles confuse search engines and users about what makes each page different.

---

### 2. Front-Load Important Keywords

**❌ Bad Example:**
```html
<title>Welcome to Our Website - The Best Digital Marketing Agency in New York</title>
```

**✅ Good Example:**
```html
<title>Digital Marketing Agency New York - SEO, PPC & Social Media Services</title>
```

**Real-Life Impact:** Amazon does this masterfully:
```html
<!-- Amazon Product Page -->
<title>Apple AirPods Pro (2nd Gen) - Active Noise Cancelling - Amazon.com</title>
```
Notice how the product name comes first, then key features, then the brand.

---

### 3. Write for Humans First, Search Engines Second

**❌ Bad Example (Keyword Stuffing):**
```html
<title>Plumber NYC, NYC Plumber, Plumbing NYC, Best NYC Plumbers, Emergency Plumber NYC</title>
```

**✅ Good Example:**
```html
<title>Emergency Plumber NYC - 24/7 Service | Quality Plumbing Solutions</title>
```

---

### 4. Match Search Intent

**Example: E-commerce Product Page**
```html
<!-- User searching: "buy nike air max 90" -->
<title>Nike Air Max 90 - Shop Men's Sneakers | Free Shipping | Foot Locker</title>
```

**Example: Informational Blog Post**
```html
<!-- User searching: "how to tie a tie" -->
<title>How to Tie a Tie: 5 Easy Methods with Step-by-Step Photos (2025)</title>
```
