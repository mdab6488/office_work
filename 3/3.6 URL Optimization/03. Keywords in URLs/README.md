# Complete Guide to Header Tags & URL Optimization

## 3.6 URL Optimization

URLs are a critical ranking factor and user experience element. A well-optimized URL tells both users and search engines exactly what the page is about.



### Keywords in URLs

Placing target keywords in your URL is a direct ranking factor and helps users understand page content.

#### Strategic Keyword Placement:

**Primary Keyword in URL:**
```
Target Keyword: "best budget smartphones"

✅ Good:
https://techreview.com/best-budget-smartphones-2025

❌ Bad:
https://techreview.com/affordable-phone-reviews
(Uses synonyms instead of target keyword)
```

**Multiple Keywords:**
```
Target Keywords: "vegan recipes", "meal prep"

✅ Good:
https://healthyblog.com/vegan-meal-prep-recipes

❌ Too Many:
https://healthyblog.com/vegan-plant-based-recipes-meal-prep-healthy-food-cooking
(Keyword stuffing)
```

#### Real-Life Examples by Industry:

**Real Estate:**
```
Target: "3 bedroom houses for sale in Austin"

✅ Optimized:
https://realestate.com/austin/3-bedroom-houses-for-sale

Alternative:
https://realestate.com/homes/austin/3-bedroom

Avoid:
https://realestate.com/listings?beds=3&city=austin&type=house
```

**SaaS Product:**
```
Target: "project management software for teams"

✅ Optimized:
https://software.com/project-management-software-teams

Alternative:
https://software.com/solutions/project-management

Avoid:
https://software.com/products/pms-v2
```

**Local Business:**
```
Target: "Italian restaurant downtown Seattle"

✅ Optimized:
https://restaurant.com/italian-restaurant-downtown-seattle

Alternative:
https://restaurant.com/locations/seattle-downtown

Avoid:
https://restaurant.com/location-3
```

#### Keyword Placement Strategies:

**1. Primary Keyword First:**
```
✅ https://site.com/keyword-other-words
❌ https://site.com/other-words-keyword
```

**2. Brand vs. Keyword:**
```
If your brand is well-known:
https://nike.com/running-shoes (brand first)

If you're building authority:
https://running-shoes-review.com/nike-pegasus (keyword first)
```

**3. Location Keywords:**
```
National targeting:
https://plumber.com/services/drain-cleaning

Local targeting:
https://plumber.com/chicago/drain-cleaning
OR
https://plumber.com/drain-cleaning-chicago
```

#### What NOT to Do:

**Keyword Stuffing:**
```
❌ https://site.com/best-pizza-nyc-best-new-york-pizza-best-pizza-delivery-nyc

✅ https://site.com/best-pizza-delivery-nyc
```

**Irrelevant Keywords:**
```
❌ https://lawfirm.com/personal-injury-lawyer-pizza-recipes
(Why is pizza there?)

✅ https://lawfirm.com/personal-injury-lawyer
```

**Forced Keywords:**
```
❌ https://blog.com/how-to-best-wordpress-seo-plugins-optimization-guide

✅ https://blog.com/best-wordpress-seo-plugins
```
