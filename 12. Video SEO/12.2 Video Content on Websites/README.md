## 12.2 Video Content on Websites

Hosting videos on your own website improves SEO, time-on-site, and conversions.

## Summary Checklist

### YouTube SEO Checklist
```
✅ Channel fully optimized (name, description, art, keywords)
✅ Titles include primary keyword + benefit (60 chars max)
✅ Descriptions detailed (200+ words, keywords, timestamps, links)
✅ 5-8 relevant tags (specific to broad)
✅ Custom thumbnails (1280x720, high contrast, minimal text)
✅ Videos organized into playlists
✅ End screens added (subscribe + related video)
✅ Cards used strategically (2-5 per video)
✅ Analytics monitored weekly
✅ CTR above 6%, AVD above 50%
```

### Website Video SEO Checklist
```
✅ Video hosting selected (YouTube/Vimeo/Wistia/self-host)
✅ Video sitemap created and submitted
✅ VideoObject schema implemented
✅ Full transcript provided on page
✅ Custom thumbnails optimized (under 100KB)
✅ Videos lazy-loaded for page speed
✅ Alt text on all thumbnails
✅ Mobile-optimized player
✅ Captions/subtitles available
✅ Related videos linked
```

---

## Key Takeaways

1. **YouTube is a Search Engine**: Optimize for discovery with keywords in titles, descriptions, tags
2. **Thumbnails Drive Clicks**: Invest time in eye-catching, high-contrast designs
3. **Watch Time is King**: Focus on audience retention over everything else
4. **Transcripts Boost SEO**: Make content indexable and accessible
5. **Schema Markup Gets Rich Results**: Implement VideoObject for enhanced visibility
6. **Analytics Guide Strategy**: Monitor CTR, AVD, traffic sources weekly
7. **Playlists Increase Session Time**: Organize content for binge-watching
8. **Page Speed Matters**: Lazy-load videos, optimize thumbnails
9. **Accessibility Wins**: Captions, transcripts, alt text help everyone
10. **Test and Iterate**: A/B test thumbnails, titles, formats continuously
