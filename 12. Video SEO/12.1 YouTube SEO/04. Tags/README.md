
### Tags

Tags help YouTube understand your video's content and surface it in related videos.

#### Tag Strategy

**Real-Life Example: Cooking Video**

```
PRIMARY TAGS (Most Important):
- carbonara recipe
- pasta carbonara
- how to make carbonara
- authentic carbonara recipe

SECONDARY TAGS:
- italian pasta recipe
- easy pasta recipe
- creamy pasta
- italian cooking

CHANNEL TAGS:
- [your channel name]
- [your name]

BROAD TAGS:
- cooking
- recipe
- italian food
- pasta recipe
- dinner ideas

LONG-TAIL TAGS:
- carbonara recipe authentic italian
- how to make carbonara without cream
- easy carbonara recipe 15 minutes
```

#### Tag Best Practices

```
✅ Use 5-8 targeted tags
✅ First tag should be exact title or primary keyword
✅ Mix specific and broad tags
✅ Include channel name as tag
✅ Use multi-word phrases (3-5 words)
✅ Add misspellings if common (e.g., "reciepe")
✅ Research competitor tags (use TubeBuddy or VidIQ)
✅ Don't use irrelevant tags (violates YouTube TOS)
```

#### Tag Hierarchy

1. **Specific Tags** (Most Important)
   - Exact match of title
   - Primary keyword variations

2. **Medium Tags**
   - Related topics
   - Category keywords

3. **Broad Tags** (Least Important)
   - General category
   - Industry terms

**Example for "iPhone 15 Pro Review":**

| Priority | Tags |
|----------|------|
| High | "iphone 15 pro review", "iphone 15 pro", "apple iphone 15" |
| Medium | "iphone review", "apple smartphone", "ios 17" |
| Low | "smartphone", "tech review", "apple" |

#### Tool Recommendations

- **TubeBuddy**: Tag suggestions, competitor analysis
- **VidIQ**: Keyword research, trend alerts
- **YouTube Search**: Check autocomplete for tag ideas
